# Фоторедактор

Простое кроссплатформенное приложение для редактирования фотографий, созданное с помощью Flutter.

## Возможности
*   Выбор изображения из галереи устройства.
*   Применение базовых настроек изображения:
    *   Яркость
    *   Контрастность
*   Предварительный просмотр изменений в реальном времени.

## Технологии
*   Flutter
*   Dart
*   `image_picker` - для выбора изображений.

## Запуск проекта
1. Убедитесь, что у вас установлен Flutter SDK.
2. Выполните команду `flutter pub get` для установки зависимостей.
3. Запустите приложение с помощью команды `flutter run`.
